# openmv-robot-vision-grip


## 桌面移动复合机器人——基于OpenMV的视觉抓取与放置系统


## 项目简介

基于 OpenMV 摄像头实现颜色识别，通过逆运动学解算控制六自由度机械臂，自动抓取黄色、橙色木块，并投放至指定区域。串口与 STM32 主控通信，完成桌面移动机器人的完整视觉抓取流程。

## 硬件平台

- 视觉模块：OpenMV Cam
- 主控制器：STM32
- 机械臂：六自由度桌面机械臂
- 通信接口：UART 串口（波特率 115200）

## 文件说明

- `main.py` —— OpenMV 主程序（颜色识别 + 逆运动学 + 串口通信）

## 运行流程

1. 上电初始化，补光灯亮起
2. 识别第一个目标（黄色木块），对准后抓取
3. 寻找粉色放置区，投放归位
4. 识别第二个目标（橙色木块），重复抓放
5. 任务完成，返回待命状态

## 开发环境

- OpenMV端：OpenMV IDE
- STM32端：Keil MDK
